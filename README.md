# Studio Élara 💄✨

> Website premium, moderno e responsivo para salão de beleza e estética, com foco em noivas e sistema de agendamento interativo integrado.

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno para entregar uma experiência visual impressionante (visual e micro-animações premium) e alta performance.

---

## 📸 Funcionalidades Principais

*   **Design Sofisticado & Premium:** Interface minimalista, paleta de cores elegantes e tipografia moderna que refletem a sofisticação do Studio Élara.
*   **Sistema de Agendamento Interativo:** Fluxo guiado passo a passo para a marcação de atendimentos:
    1.  **Seleção de Serviço:** Escolha entre Cabelo, Maquiagem, Unhas ou Pacote Noiva.
    2.  **Seleção de Data e Horário:** Calendário visual interativo com horários disponíveis.
    3.  **Formulário de Contato:** Coleta de dados básicos (nome, telefone, observações).
    4.  **Confirmação/Sucesso:** Tela de sucesso com resumo da reserva.
*   **Espaço Noivas:** Seção dedicada destacando pacotes exclusivos e momentos especiais para o "Dia da Noiva".
*   **Galeria de Portfólio:** Grid de fotos otimizado demonstrando os melhores trabalhos do salão.
*   **Equipe de Especialistas:** Seção apresentando os profissionais do Studio Élara.
*   **Depoimentos:** Feedback real de clientes com avaliações e fotos de antes/depois ou resultados.
*   **Acessibilidade & Contato Direto:** Botão flutuante do WhatsApp para suporte imediato e cabeçalho/rodapé com links rápidos.

## 🛠️ Tecnologias Utilizadas

*   **Framework:** [Next.js 15+](https://nextjs.org/) (React 19)
*   **Estilização:** [Tailwind CSS](https://tailwindcss.com/)

*   **Ícones:** [Lucide React](https://lucide.dev/)

## 🚀 Como Iniciar o Projeto

### Pré-requisitos

Certifique-se de ter instalado em sua máquina:
*   [Node.js](https://nodejs.org/) (versão 18 ou superior recomendado)
*   [npm](https://www.npmjs.com/) ou outro gerenciador de pacotes (yarn, pnpm, etc)

### Instalação

1. Clone este repositório:
   ```bash
   git clone https://github.com/DuardoHenrique/salao-de-beleza.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd studio-élara
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

### Execução em Desenvolvimento

Inicie o servidor local de desenvolvimento:
```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

### Produção

Para gerar a build de produção e iniciar o servidor otimizado:
```bash
# Criar build de produção
npm run build

# Iniciar servidor
npm run start
```
